Cyber Security Analyst #CSA71 $60,000 – $95,000 Full-Time Remote Reports To: Information Security Manager

Monitor, detect, and respond to security threats, ensuring the confidentiality and integrity of organizational systems.

Key Responsibilities

  • Monitor SIEM dashboards and detect unusual activity.
  • Conduct vulnerability scans and prepare mitigation reports.
  • Perform security audits on networks and endpoints.
  • Investigate incidents, malware alerts, and breaches.
  • Maintain firewall policies and endpoint security tools.

Skills and Qualifications

Required Skills Preferred Skills
  1. 1–3 years experience in cybersecurity.
  2. Knowledge of SIEM tools (Splunk, QRadar).
  3. Strong understanding of OS security (Windows/Linux).
  4. Good knowledge of networking concepts (TCP/IP, ports, firewalls).
  1. Experience with penetration testing tools (Burp Suite, Nmap).
  2. Knowledge of cloud security (AWS IAM, Azure Defender).
  3. Incident response workflow knowledge.
  4. Certifications: CompTIA Security+, CEH, or equivalent.

Network Administrator #NAD88 $50,000 – $80,000 Full-Time Local Reports To: IT Infrastructure Manager

Install, manage, and support network systems to ensure seamless connectivity across the organization.

  • Configure and maintain routers, switches, and firewalls.
  • Troubleshoot network outages, latency, and performance issues.
  • Monitor network traffic and set up alerts.
  • Maintain VPN and user access controls.
  • Install and update network-related hardware and firmware.

Skills and Qualifications

Required Skills Preferred Skills
  1. 4+ years experience in network administration.
  2. Knowledge of TCP/IP, routing, switching, subnetting.
  3. Experience with Cisco or Juniper devices.
  4. Understanding of firewalls and VPNs.
  1. Knowledge of Linux/Windows servers.
  2. Experience with network monitoring tools (Nagios, Zabbix).
  3. Basic scripting knowledge (Bash/Python).
  4. Certification: CCNA, Network+.
